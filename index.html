Tabletop Game Tracker

Welcome to the Tabletop Game Tracker, this fun little app will help you and your friends track your tabletop game sessions by keeping a record of each players choices, and the resulting dice rolls.

To use this application, click on “START GAME.” The tracker will then begin presenting the nessacary prompts for entries. If you wish to print your game data, please select “PRINT.”

How to use this application:

To use this application, click on “START GAME.” The tracker will then begin presenting the nessacary prompts for entries. If you wish to print your game data, please select “PRINT.”


Application Features:

Game Tracking

Tracks player choices and the resulting dice rolls.

Printer Friendly

If you decide to print off your data, you can rest assured that it will print off in an easy on the eyes format that strives to be easy on your ink supply as well.

Custom Role Modifiers

The application contains the “natural” modifier by default, but you can also add custom modifiers to suit your game, edition and overall paly style.


Dice Style Options

This application contains logic to simulate the rolling of a D20 dice.

Mobile Responsive

Use the app on your laptop, tablet or phone this application was designed with mobile in mind.

User Accounts

Can't or don't want to print off your data? Thats fine, simply create or log in to your account to save your data and view past games.

Things to note:

At this time this application only rolls a D20
If you accidentally exit the game prompts, you may have to start a new instance of the application by clicking “START GAME.” Strongly consider printing out your current data before stating a new game to help keep track of your session. We hope to be able to avoid this issue in the future.
At this time the user accounts feature is unavailable, please print your data if you need to save it until the accounts feature is available. Thank you.
If you run into any issues while using the application, please proceed to "git repo link" and leave a comment clearly describing the problem. Please also include any error codes you may have received.
